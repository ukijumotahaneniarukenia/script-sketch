sdkコマンドは実行者単位でインストールされる

```
yum install -y zip

curl -s "https://get.sdkman.io" | bash
```

特定のコマンドを実行すると最新バージョンでインストールされる

これも実行者単位でインストールされる

```
sdk install gradle
```

```
/root/.sdkman/candidates/gradle/6.4.1/bin/gradle
```

一度ダウンロードしたらアーカイブされる

これも実行者単位

```
/root/.sdkman/archives/gradle-6.4.1.zip
```
