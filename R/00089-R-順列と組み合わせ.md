Rで組み合わせなど

- https://www.rdocumentation.org/packages/gtools/versions/3.5.0

- https://www.rdocumentation.org/packages/gtools/versions/3.5.0/topics/combinations


```
sudo Rscript -e 'install.packages("gtools",repos="http://cran.ism.ac.jp/", dependencies = TRUE, Ncpus = getOption("Ncpus", 12L))'


重複なし組み合わせ

echo 2 {A..D} | Rscript -e 'options(max.print=2147483647);suppressMessages(library(gtools));s<-readLines("stdin");a<-unlist(strsplit(s," "));b<-a[1];c<-a[2:length(a)];r<-combinations(length(c),ifelse(as.integer(b)>length(c),length(c),max(as.integer(b),1)),c,repeats=FALSE);write.table(r,quote=FALSE,row.names=FALSE,col.names=FALSE,,sep="\t",stdout())'
A	B
A	C
A	D
B	C
B	D
C	D
```
