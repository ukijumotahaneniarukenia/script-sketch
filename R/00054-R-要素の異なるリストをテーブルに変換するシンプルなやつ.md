- この形式をテーブルにする
```
[[1]]
[1] "ウ" "ン" "コ" "A"  "B"  "C"

[[2]]
[1] "も" "り" "も" "り" "A"  "B"  "C"

[[3]]
[1] "森" "鴎" "外" "A"  "B"  "C"
```

- IN
```
$parallel echo ::: {ウンコ,もりもり,森鴎外} ::: ABC | ./a.R
```

- CMD

```
#!/usr/bin/env Rscript

s<-readLines("stdin");
ss<-strsplit(s," ");
sss<-sapply(ss,function(e){
  return(
          c(
            unlist(strsplit(e[1],""))
           ,unlist(strsplit(e[2],""))
          )
        )
})

names(sss)<-c(1:length(sss))

stack(sss)
```
