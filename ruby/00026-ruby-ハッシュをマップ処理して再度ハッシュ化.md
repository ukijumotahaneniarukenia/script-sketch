
- CMD

```
ruby -e 'h = { "apple" => 100, "orange" => 200, "grape" => 300 };p h.map { |key, value| [key, value * 2] }'
```

- OUT

```

[["apple", 200], ["orange", 400], ["grape", 600]]

```


- CMD


```

ruby -e 'h = { "apple" => 100, "orange" => 200, "grape" => 300 };p h.map { |key, value| [key, value * 2] }.to_h'

```

```

{"apple"=>200, "orange"=>400, "grape"=>600}

```
