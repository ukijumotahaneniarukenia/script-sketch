- CMD
  - ebanさんのperlとオプション引数の解釈は似てる
```
$echo caffelatte | ruby -nle 'p $_.chars.group_by{|key|key}.map{|liz|[liz[0],liz[1].size]}'
```

- OUT

```
[["c", 1], ["a", 2], ["f", 2], ["e", 2], ["l", 1], ["t", 2]]
```

- CMD

  - なにか一文字
```
$ echo caffelatte | ruby -F'\\?' -anle '$F.group_by{_1}.sort.map{|k,v|p k,v.size}'|xargs -n2
```

- OUT

```
a 2
c 1
e 2
f 2
l 1
t 2
```
