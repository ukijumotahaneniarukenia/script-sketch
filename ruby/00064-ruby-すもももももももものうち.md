- CMD
  - $0はデフォで標準出力に出力されるので、埋め込んでいる $2$1を各入力行単位で。

  - $0は結果の蓄積の働きとして利用

```
echo すもももももももものうち | grep -o . | uniq -c | awk '$0=$2$1' ORS=
```

- OUT

```
す1も8の1う1ち
```
