-eオプションの直後にスペースあけると動かない
```
$echo 響け！ユーフォニアム | awk -e '{print substr($0$0,'{1..10}',11)}'
awk: 致命的: ファイル `{print substr($0$0,2,11)}' を読み込み用に開けません (No such file or directory)
```
