
- REF

- IN

  - \x1Bはエスケープ\e(ESC)のこと

  - \eはキーボードのESCキー

  - \eの代わりに\033（8進数形式のESC）や\0x1b、\0x1B、\x1b、\x1B（どれも16進数形式のESC）も可

```
printf '\e[31mhoge\e[m\n' | sed -r "s/\x1B\[([0-9]{1,2}(;[0-9]{1,2})?)?[m|K]//g"
```

- OUT

```
hoge
```

- CMD

  - ヘルプが色付けされていたら色抜き

```
trans -h | sed -r "s/\x1B\[([0-9]{1,2}(;[0-9]{1,2})?)?[m|K]//g"
```
