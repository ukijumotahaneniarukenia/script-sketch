- CMD

  utf8入るとグルーピングがめんどいので、ミュート

```
$ echo  -ne ನ್  | uniname -ceb 2>/dev/null | jq -Rcr './"\\n"|map(split(" ")|map(select(test("[^\u0020]"))))|transpose'
```

- OUT

```
[["UTF-32"],["glyph"],["name"]]
[["000CA8"],["ನ"],["KANNADA"],["LETTER"],["NA"]]
[["000CCD"],["್"],["KANNADA"],["SIGN"],["VIRAMA"]]
```


- CMD

```
$ echo  -ne ನ್  | uniname -cb 2>/dev/null | jq -Rcr './"\\n"|map(split(" ")|map(select(test("[^\u0020]"))))|transpose'
```

- OUT

```
[["UTF-32"],["encoded"],["as"],["glyph"],["name"]]
[["000CA8"],["E0"],["B2"],["A8"],["ನ"],["KANNADA"],["LETTER"],["NA"]]
[["000CCD"],["E0"],["B3"],["8D"],["್"],["KANNADA"],["SIGN"],["VIRAMA"]]
```

- CMD

```
$ echo  -ne ನ್  | uniname -ceb 2>/dev/null | jq -Rcr './"\\n"|map(split(" ")|map(select(test("[^\u0020]"))))|transpose as $a|($a[0],$a[1],[$a|.[2:($a|length)]|flatten|join("-")])'|xargs -n3
```

- OUT

```
[UTF-32] [glyph] [name]
[000CA8] [ನ] [KANNADA-LETTER-NA]
[000CCD] [್] [KANNADA-SIGN-VIRAMA]
```
