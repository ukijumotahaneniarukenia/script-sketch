- CMD
  - reduceをつかう

```
$ cat c.json | jq 'reduce .[] as $item({};.+$item)'
```

- OUT

```
{
  "col0": "202-4708",
  "col1": "斎藤 七海",
  "col2": "合資会社小川鉱業",
  "col3": "Orchestrator",
  "col4": "Contract",
  "col5": "Master"
}
```
