- IN


```
$ echo あいう203| jq -R 'split("")'
[
  "あ",
  "い",
  "う",
  "2",
  "0",
  "3"
]
```

- CMD

```
$ echo あいう203| jq -R 'split("") as $a |$a|map(select(test("[\u3042-\u3049]")))'
```


- OUT

```
[
  "あ",
  "い",
  "う"
]
```
