
```
$cat input.json
[
  {
    "id": "abc",
    "name": "name-middlenane-lastname-1"
  },
  {
    "id": "123",
    "name": "fname-flast-2"
  }
]
```

- CMD
```
$jq 'map(. + (.name | capture("(?<newkey>.+)-[0-9]+")) )' input.json
```


- OUT
```
[
  {
    "id": "abc",
    "name": "name-middlenane-lastname-1",
    "newkey": "name-middlenane-lastname"
  },
  {
    "id": "123",
    "name": "fname-flast-2",
    "newkey": "fname-flast"
  }
]
```
