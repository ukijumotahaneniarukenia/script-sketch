#複数オブジェクトを単一オブジェクトに変換

- CMD
  - slurpからのfrom_entriesでいける
```
cat l.json | jq --stream -c 'select(length==2)|{key:.[0]|join("."),value:.[1]}' | jq -s 'from_entries'
```
OUT
```
{
  "a": "foo",
  "b.c.d.test.key": "value"
}
```
