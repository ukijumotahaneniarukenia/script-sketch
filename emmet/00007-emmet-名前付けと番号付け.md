- IN

```
$ echo 'ul>li.sample$*5'
ul>li.sample$*5
```


- CMD


```
$ echo 'ul>li.sample$*5' | emmet
```

- OUT

```
<ul>
	<li class="sample1"></li>
	<li class="sample2"></li>
	<li class="sample3"></li>
	<li class="sample4"></li>
	<li class="sample5"></li>
</ul>
```

- CMD

```
$ echo 'ul>li.sample$*5' | emmet | jq -Rr '[splits("[^0-9a-zA-Z]\\K")|splits("[0-9a-zA-Z]+\\K")|select(""!=.)]|join("")'
```


- OUT

```
<ul>
	<li class="sample1"></li>
	<li class="sample2"></li>
	<li class="sample3"></li>
	<li class="sample4"></li>
	<li class="sample5"></li>
</ul>
```
