指定した一般ユーザーディレクトリ配下の資産をWEBサーブする

サーブディレクトリの作成

```
mkdir -p $HOME/public_html
mkdir -p $HOME/public_html/test1
mkdir -p $HOME/public_html/test2

echo うんこ > $HOME/public_html/index.html

echo aaaaaaaaaaa > $HOME/public_html/test1/a.html

echo bbbbbbbbbbb > $HOME/public_html/test2/b.html
```

serverディレクティブに追加

修正内容

$1がユーザー名
$2がグロブ展開（配下すべて）

インデックスページのエイリアス

キャッシュ自動作成するためのnginxプロセス監視対象に追加
```
location ~ ^/~(.+?)(/.*)?$ {
	alias /home/$1/public_html$2;
	index  index.html index.htm;
	autoindex on;
}
```

設定ファイルをjsonで出力したい

配下すべてグロブ効くので、まるごと入れてブラウザで見るのもあり。ブラウザの拡張機能も使えたりするので。

```
$ cp -r $HOME/script-dat $HOME/public_html/
$ cp -r $HOME/script-cmd $HOME/public_html/
$ cp -r $HOME/script-crawler $HOME/public_html/
$ cp -r $HOME/script-sketch $HOME/public_html/
```

たとえばjsonやxmlファイルもブラウザでみると色付けされて確認できたりする

```
firefox http://localhost/~aine/script-dat/test.xml
firefox http://localhost/~aine/script-dat/test.json
```
