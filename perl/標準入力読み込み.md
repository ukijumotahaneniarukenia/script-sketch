## 単一行複数列読み込み
```
$echo {a..d} | ./標準入力読み込み.pl
a b c d
```

## 複数行複数列読み込み
```
$echo {a..g} | xargs -n4 | ./標準入力読み込み.pl
a b c d
e f g
```
