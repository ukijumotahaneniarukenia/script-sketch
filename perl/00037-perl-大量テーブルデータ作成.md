  - qq演算子はエスケープ対策用だと思われ
  - 1000万件
```
time perl -MTime::Piece -MPOSIX -e 'printf(qq/%05d %05d %d %d %s\n/,ceil ($_/3),$_,$_%3,int(rand 10)+1,$t=localtime->datetime) for 1..10000000' | awk '4'
```
