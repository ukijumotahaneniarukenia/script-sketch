コードポイント0とコードポイント1114112以外のスタートとエンドを取得

```
$ perl -C -MUnicode::UCD -E '$,="\t";($i,$r)=Unicode::UCD::prop_invmap("Zs");@a=@{$i};map{say @{$i}[$_],@{$r}[$_]}(0..$#a)'|awk -v FS="\t" '$2=="N"{print 0,$1,$2}$2=="Y"{print 1,$1,$2}'| awk '{print a=a+$1,$2,$3}'|awk '($1!=0&&$2!=0)' | awk '($2!=1114112)' | awk 'BEGIN{SEP="\t"}{a[$1]=a[$1]SEP$2}END{for(e in a)print e,a[e]}'|sed -r 's/ +//g'|sort -nk1
```
