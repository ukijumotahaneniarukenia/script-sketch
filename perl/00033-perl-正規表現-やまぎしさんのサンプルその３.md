- CMD
  - 全量マッチ
```
$cat test3.txt | perl -nlE '// and say'
```

- OUT
```
ABCXDXEFXGHIJ
ABCXXDEFXGHXIJ
XABCDEXFXGHXIJ
XXDEFXGHXIJ
XXXX
```

- CMD

```
$cat test3.txt | perl -nlE 's/(^[^X]*X)(.*)/\1/ and say'
```


- OUT

```
ABCX
ABCX
X
X
X
```

- CMD

```
$cat test3.txt | perl -nlE 's/(^[^X]*X[^X]*X)(.*)/\1/ and say'
```


- OUT

```
ABCXDX
ABCXX
XABCDEX
XX
XX
```

- CMD

```
$cat test3.txt | perl -nlE 's/(^[^X]*X[^X]*X[^X]*X)(.*)/\1/ and say'
```

- OUT

```
ABCXDXEFX
ABCXXDEFX
XABCDEXFX
XXDEFX

```

- CMD
  - 行頭は１つしかないので、マッチできる

```
$cat test3.txt | perl -nlE 's/((?:^[^X]*X){1})(.*)/\1/ and say'
```

- OUT

```
ABCX
ABCX
X
X
X
```

```
$cat test3.txt | perl -nlE 's/^(([^X]*X){1})(.*)/\1/ and say'
```
