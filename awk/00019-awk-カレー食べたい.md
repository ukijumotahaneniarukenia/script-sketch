- CMD
  - aが結果の蓄積の役割
  - デフォで$0を出力するので、$0=$0aで今回の蓄積と前回までの蓄積を標準出力に出力している
  - a=($0=$0a)で今回までの結果の蓄積を変数aにホールドし、次の行を処理
  - ebanさんの
```
echo カレー食べたい | grep -o . | awk 'a=$0=$0a'
```

- OUT


```

レカ
ーレカ
食ーレカ

```
