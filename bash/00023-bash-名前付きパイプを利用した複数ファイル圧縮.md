複数ファイルバージョン

バッググラウンド実行は最後に持ってくるイメージ

```
$time ls dump* | while read f;do mkfifo $f-in-pipe;cat $f > $f-in-pipe;{ cat $f-in-pipe | gzip -c >$f.gzip; } & done
```
