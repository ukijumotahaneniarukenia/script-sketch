#!/bin/bash

```
for file in $@; do
  vim -X -N -u NONE -i NONE -V1 -e --cmd "source $file | qall!" 2>&1
done
```

```
-X          X サーバーと通信しません。端末での起動時間を短くできます。 しかし、ウィンドウタイトルの変更やクリップボードは使えなくなります。

--noplugin  プラグインをロードしません。-u NONE はこの動作を含んでいます。

-N          非互換モード。'compatible' オプションがオフになります。
            .vimrc ファイルの有無に関わらず、 Vim の改良された機能が有効になります。Vi との互換性が少し失われます。


-V[N]       冗長モード。 スクリプトファイルを実行したり viminfo ファイルを読み書きするたびにメッセージを表示します。 N に指定した数値が 'verbose' に設定されます。 省略した場合は 10 になります。


-e          Ex モードで起動します。 実行ファイルの名前が "ex" の場合と同じです。

--cmd {command}
            "-c" と同じですが、vimrc を処理する前にコマンドが実行されます。 これらのコマンドは "-c" コマンドとは別に、10 個まで指定できます。


:qa[ll]!	  Vimを終了する。バッファの変更点は全て失われる。
		        終了コードを非ゼロにしたい場合は |:cquit| を使う。

							*:quita* *:quitall*

```
