- CMD

  - json形式でハンドリングする際、jqコマンドはコメントがあると実行エラーになるので、除去しておき、xmllintコマンドで名前空間ありのXMLファイルをハンドリングすることになると、記述量が多くなるので、前処理で除去
```
cat pom.xml | xml c14n --without-comments - | xmllint --format - | teip -Gog '(?<=<project).*?(?=>)' -- sed 's/.//g' | xmllint --format - >pom-remove-comment-namespace.xml
```
